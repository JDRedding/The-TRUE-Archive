===============================================================================
1. PURPOSE
===============================================================================
The TRUE Archive is a cosmic-scale classification engine, metaphysical library,
and exploration framework. It provides:

- A unified taxonomy for all realities (physical, extreme, meta, annex).
- A dynamic operator system (dot notation) for activated states.
- A playable structure for exploration, risk, and discovery.
- A metaphysical OS (MANA OS) for rituals, Essence flow, and interstratal travel.
- A social ecosystem of Lodges, Guilds, and Curator-driven interactions.

3. THE ANNEX CODE SYSTEM
===============================================================================

FORMAT
    [CLASS / SUBCLASS – REALITY – DIVERGENCE – SUBTYPE]

CLASS (A–Ω)
    A   = Proto-habitations
    M   = Familiar, terrestrial
    A̱   = Mythic, primordial
    Æ   = Magic, ritual-dominant
    X   = Xenorealities
    U–Z = Meta-realities
    Ω   = Annex / Anti-reality

SUBCLASS (00–99)
    Logarithmic proximity to baseline.
    00 = identical
    01–10 = minor changes
    11–30 = moderate
    31–60 = major
    61–90 = radical
    91–99 = unrecognizable

REALITY (00–99)
    Environmental / structural characteristics.

DIVERGENCE (00–99)
    Deviation from expected physics/logic.

SUBTYPE (a–z / 0–9)
    Thematic or cultural flavor.

EXAMPLES
    [M/00-00-00-a]     Earth-like, Newtonian, steampunk
    [A̱/05-80-30-b]     Mythic, high magic, post-apocalyptic
    [Ω/XX-99-99-Ω]     Annex, undefined, max divergence

4. STATIC vs DYNAMIC CLASSES (DOT NOTATION)
===============================================================================

STATIC CLASS
    M, A̱, Æ, X, U, V, W, Y, Z, Ω
    → What a reality IS.

DYNAMIC CLASS (˙)
    Ṁ, Ȧ̱, Æ̇, Ẋ, U̇, V̇, Ẇ, Ẏ, Ż, Ω̇
    → What a reality is DOING.

THE DOT (˙) = OPERATOR
    Converts:
        category → operation
        definition → execution
        state → process
        record → behavior

REQUIRED FOR:
    - MANA OS runtime
    - Ritual operations
    - Essence-flow activation
    - Interstratal propagation
    - Relational derivatives

ASCII TABLE
    Static (M)        | Dynamic (Ṁ)
    ------------------|-------------------------
    Passive           | Active
    No Essence Flow   | Channeling
    Data              | Runtime
    Target            | Operator
    Contained         | Propagating
    Planet            | Planet mid-teleport
