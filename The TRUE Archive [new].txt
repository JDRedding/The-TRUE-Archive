The TRUE Archive: The archivists and expeditionaries document realities from the past, present, and future of countless universes for the benefit of visitors.

The Annex Code System: The collection is organized by alphanumeric codes that indicate how similar a universe reality [A-00-00-00a] to another. Codes closer to this sequence represent more familiar worlds, while further ones represent more alien or incomprehensible realities. The TRUE Archive is a rich historical world-building database and library of cosmic scale, curated mystery, and interactive exploration. 

The TRUE Archive: Core Structure
Purpose & Theme
- Cosmic TRUE Library: A trans-dimensional multidimensional repository where every possible (and seemingly impossible) reality, story, timeline, and existence is documented as a "record" (e.g., a book, hologram, or living memory).
- Philosophy: "The past is a story, we, historians, tell ourselves. The future is what we record. The Archive holds all."
- Tone: Mystical yet hard historical science that is systematic.

Experience: Entities are invited to wander the galleries and observatories to witness stories unfolding in other realms.

Interaction: The curator invites visitors to ask questions if they seek specific information, which will then be retrieved by the archivists.

Final: Entities are encouraged to keep an open mind, as the records provided are intended to be starting points for journeys that only the receiver can embark upon.

The Annex Code System: Expanded
Code Anatomy
current format: `[A/0-00-00-00.0]`
Hybrid hexadecimal hierarchical expansion for TRUE compatibility:

```
[CLASS-REALITY-DIVERGENCE-SUBTYPE]
```
- CLASS (0-Ω):
  - `M`: Familiar (e.g., Earth-like, D&D settings).
  - `A̱`: Mythic (e.g., gods, magic-dominant).
  - `X`: Xenorealities (e.g., non-Euclidean, time-looped).
  - `Ω`: Omega (realities that defy classification; Annex-only).
- SUBCLASS (/0): Proximity to a "baseline" (e.g., `00` = our universe, `01` = minor changes, `99` = unrecognizable), log scale based on user.
- REALITY (00-99): General characteristics.
- DIVERGENCE (00-99): Degree of deviation from expected physics/logic (e.g., `00` = Newtonian, `50` = magic exists, `99` = reality is a song).
- SUBTYPE (a-z/0-9): Specific flavor or theme (e.g., `a` = steampunk, `b` = post-apocalyptic, `1` = cyberpunk).

The Archive classifies locations by lettered codes based on characteristics, like habitability, atmosphere, surface conditions, and proximity, with Class M being the most suitable for humanoid life.

Overview of Classes

Class #–9: Interstratal locations
Locations situated “between” ... that is, spaces that exist in the perceptual and cognitive domain rather than in a concrete geographic reality. These "inner" planes function as foundational, elemental realms that are perceived as central to the multiverse. Inner planes are the raw substrate of matter and energy in cosmologies. They surround or underlie the Material Plane (representing the "normal" world) and provide the elemental and energetic foundations upon which all other layers of a multiverse are constructed.

Class A–F: Proto-Habitations
Class A: Geothermal, surfaces partially molten, mostly hydrogen-based atmospheres, no life, located in cold or ecosphere zones. 
Class B: Hot with tenuous atmospheres, surfaces partially molten, uninhabitable. 
Class C: Geologically inactive, cold with frozen atmospheres, generally uninhabitable. 
Class D: Small, heavily cratered or barren, almost no atmosphere; examples include moons or asteroid-like planetoids. 
Class E: Newly formed in habitable zones with molten surfaces, leading eventually to Class F. 
Class F: Partially solidified crusts, some hydrogen compounds in atmosphere, evolving toward Class G, generally uninhabitable by humanoids.

Class G–L: Marginal to Partially Habitable
Class G: Crystallizing surfaces, toxic atmospheric gases, potentially habitable with technological aid; may eventually become Classes K–P. 
Class H: Hot, arid with heavy gases or metallic vapors, largely uninhabitable for humanoid life but some extremophile forms possible.
Class I: Ice or gas “superlocations,” with ammonia/water/methane atmospheres, primarily uninhabitable.
Class J: Gas-like locations, uninhabitable for humanoids, sometimes with layered atmospheres that could theoretically support non-humanoid life.
Class K: Adaptable planets, potentially habitable for humanoids with life support measures, often pre-terraforming state 
Class L: Marginally habitable, rocky planets with thin breathable atmospheres; can support limited vegetation or colonies with technological aid

Class M–P: Fully or Partially Habitable
Class M: Terrestrial, oxygen-nitrogen atmospheres, significant liquid water, abundant vegetation and animal life; Earth is the prime example.
Class N: Highly reducing with dense, greenhouse gas-heavy atmospheres; very hot, generally uninhabitable.
Class O: Pelagic location, mostly covered with liquid water (>80%), suitable for aquatic life, similar to Class M but more oceanic.
Class P: Glaciated, >80% ice coverage, hardy lifeforms possible, sometimes considered Class M or O under warmer conditions.

Class Q–Y: Extreme or Variable 
Class Q: Variable or unstable locations due to eccentric fluctuations; environmental conditions range from molten to frozen. 
Class R: Rogue location in space or dimensional; may generate limited geothermal ecosystems. 
Class S–T: Gas ultra-locations or hot locations with enormous gravity, radiating heat (from locational cores), mostly uninhabitable.  

CLASS U–Z : Meta-reality
These classes fill the gap between Class Q–Y (extreme physical) and Class Ω (Annex/Nightmare). They represent non‑physical, semi‑physical, or supra‑physical environments that expeditionaries can enter, observe, or become trapped in. Each class begins with a class variable.

Class U : 🜄 : Unstable Realities; Meta‑variable, semi‑coherent; Transitional, collapsing, or partially‑rendered universes; Physics loads only when observed; collapses when unobserved; Time is probabilistic, not sequential; Geography “shimmers” between states; Expeditionaries risk Echo‑Shear (temporal desync); A universe mid‑boot, or a corrupted save file

Type: Harmonic, frequency‑dominant

Characteristics

General Notes
Classes consider age, location, surface conditions, atmosphere composition, and potential for life. Classes can evolve as age geologically: e.g., Class A→C→G→M in some cases. Class M is canonically the most referenced as suitable without special modifications. Other dangerous or extreme classes like Y or Q are often flagged to navigate hazardous scenarios or unique cultures. These classifications are utilized across the archive to give scribes and expeditionaries a consistent way to understand environments, whether for exploration, colonization, or potential hazards in missions.

Institutional Orders of the Archive: Lodges & Guilds

🜁 Lodges & Guilds of the TRUE Archive
---
(Institutional Factions, Expeditionary Orders, and Para‑Scholastic Societies)
The Archive is not merely a place — it is an ecosystem of competing, cooperating, and knowledge‑orders. These Lodges and Guilds form the social, political, and metaphysical infrastructure that keeps the Archive functioning (or malfunctioning).

1. The Lodge of Silent Binding

			“We bind what must not wander.”

Role: Prevent dangerous records from escaping their containers.
Specialty: Metaphysical sealing, anti‑contamination wards, Annex quarantine.
Perk: Members can “mute” a record’s Echoes for short periods.
Cost: Gradual loss of personal memories (the Lodge calls this “donation”).

2. The Lodge of the First Lens

			“Observation is creation.”

Role: Overseers of Observatories; experts in high‑Divergence study.
Specialty: Sanity/Essence‑preserving viewing protocols, lens‑geometry, observer‑paradox mitigation.
Perk: Can view Annex‑tier realities without immediate rupture.
Cost: Members slowly become partially transparent.

3. The Lodge of the Curator’s Shadow

			“The Curator sees all. We see what it does not.”

Role: Internal auditors, spies, and counter‑oracles.
Specialty: Tracking Curator anomalies, predicting Archive moods, sabotage prevention.
Perk: Access to Curator‑locked wings.
Cost: The Curator always knows their names.

🜄 The Guilds (External‑Facing Expeditionary Orders)
---
Guilds are outward‑facing, sending members into other universes to retrieve, repair, or record. They operate like cosmic unions: each has ranks, rites, and hazard pay.

1. Guild of Echo Divers
------------------------

			“We swim in the wake.”

Role: Enter unstable timelines to capture Echoes (psychic imprints).
Tools: Echo‑nets, mnemonic anchors, divergence‑tuning forks.
Risk: Echo‑overload → personality fragmentation.
Prestige: Highest fatality rate = highest respect.

2. Guild of Annex Cartographers
--------------------------------

			“The Annex is not a place. It is a direction.”

Role: Map the unmappable; chart Ω‑class realities.
Tools: Topological compasses, paradox sextants, recursive maps.
Risk: Returning with a map that maps you instead.
Prestige: Only guild allowed to petition the Curator directly.

The Archive's Cartographers are among the most respected (and quietly feared) expeditionaries in the Archive. While most guilds venture into known or semi-known realities, the Cartographers alone are sanctioned to push into Ω-class territories — realities that actively resist classification, where the Archive’s own codification systems begin to fail. They do not “explore” in the traditional sense. They witness topology and attempt to bring back something coherent enough to be recorded without unraveling the mind of the reader.

Guild of Annex Cartographers: Rank StructureThe Cartographers maintain the same foundational ranks used across all Lodges and Guilds of the TRUE Archive, but each rank carries specialized meaning, privileges, and dangers within their Ω-focused order.

1. Initiate
“Eyes only. Hands bound.”  Newcomers who have proven mental resilience but no field experience.  
Permitted to study sanitized Ω-class summaries in secure Observatories.  
Forbidden from touching any Annex artifact or entering even Class U–Z realities.  
Common Trial: Survive 72 hours observing a single Recursive Map without losing object permanence.

2. Scribe

		“We record what should not be seen.”   

Trusted with cataloging low-to-mid divergence Echoes brought back by higher ranks.  
Allowed limited access to Class W and X records under supervision.  
Begin training with basic Topological Compasses and Echo-Veils.  
Many Scribes spend years redacting their own memories after their first true exposure.

3. Expeditionary

			“We walk the direction that should not exist.” 

The backbone of the Guild. Authorized for active fieldwork in Class U–Z and select Ω-fringe realities.  
Issued standard toolkits: Topological Compass, Paradox Sextant, and basic Echo-Veil.  
Expected to return with at least one viable map fragment per mission.  
Highest attrition rate of any Guild rank. Survivors often bear visible topological scars (skin that folds unnaturally).

4. Chronicler

			“We make the unmappable... legible.”  

Elite record-keepers who translate Annex experiences into Archive-compatible formats.  
Can alter minor records and stabilize small timeline fractures.  
Granted access to the Null Quill and advanced Recursive Maps.  
Many develop the ability to “speak in glyphs” — communicating complex Ω-topologies without words.

5. Warden

			“We guard the threshold so others may cross it.”  

Senior field commanders who oversee Annex access gates and major mapping expeditions.  
Responsible for quarantining cognitohazards and training new Expeditionaries.  
Carry personalized artifacts (often a fully sentient Recursive Map or custom Paradox Sextant).  
Authority to seal or unseal specific Ω-class wings.

6. Prime

			“The Map and the Cartographer become one.”  

There is only one Prime per Guild at any time.  
The current Prime of the Annex Cartographers is known only as The Folded Cartographer (real name redacted even from most Wardens).  
Negotiates directly with the Curator. Holds the master key to the deepest Annex vaults.  
Legend says the Prime no longer has a fixed form — they exist as a living topological anomaly that walks between realities.  
Final Privilege: The ability to declare an entire reality “Unmappable” and request its excision from the Archive.

Primary Tools of the Annex Cartographers
----
Here are the most notable instruments they employ. Each is both artifact and semi-sentient protocol:

Topological Compasses
--
These do not point north — they point toward least resistance in essence geometry.  Appearance: A brass-and-crystal sphere filled with slowly rotating liquid starlight and tiny moving glyphs.  
Function: Detects folds, self-intersections, and causal knots in reality. It can show the “shortest path” through a looping timeline or a reality that has been crumpled like paper.  

Risk: Prolonged use may cause the user to perceive all straight lines as lies. Some Cartographers return unable to walk in Euclidean space without vertigo.  

Guild Rank Required: Expeditionary or higher.

Mechanics:
Short Range (100m)	: Reveals safe paths (⟐), unstable routes (⟁).
Long Range (1km)	: Detects Annex proximity (⧉).
Overuse				: User perceives all lines as curved (lasts for days).
Critical Failure	: User becomes a non-Euclidean entity (permanent).

Paradox Sextants
--
Intricate multi-ringed devices that measure angles between contradictory truths.  Function: Allows the user to triangulate their position using three mutually impossible reference points (e.g., “before the beginning,” “after the end,” and “the place that never was”).  
Special Ability: Can temporarily stabilize a collapsing Ω-reality long enough to take a single reading or extract one Echo.  
Drawback: Each successful use slightly increases the probability that the user becomes a paradox themselves.

Mechanics:
Success				: Stabilizes one Ω-reality for minutes.
Failure				: User gains a paradox trait (e.g., "exists in two places at once").
Critical Failure	: User becomes a paradox (discretion).

Recursive Maps
--
The signature tool of the Guild. These are not static charts.  Form: Usually a scroll, a crystal lattice, or a living tattoo that grows across the skin.  
Property: The map redraws itself as new information is perceived. However, the map sometimes begins mapping the Cartographer — their thoughts, fears, or alternate selves bleeding into the geography.  
Advanced Technique: “Folding the Map” — deliberately collapsing the representation to force a corresponding collapse (or expansion) in the target reality.

Mechanics:
Passive: Maps current reality.
Active (Folding the Map): Collapses or expands target reality (requires Warden approval).
Risk: Map begins mapping the user (1% chance per use).

Echo-Veils & Mirror Shrouds
--
Thin, shimmering membranes worn as cloaks or placed over the eyes.  
Purpose: Filter out cognitohazards and prevent the Annex from “looking back” too directly at the observer.  
Limitation: The longer one wears a Veil, the more difficult it becomes to remember one’s original face.

The Null Quill
--
A writing instrument, a meta-stable mechanism (Quills have a limited lifetime of existence), that is manufactured or produced only when needed.  
Used to inscribe observations that cannot be spoken or thought directly.  
Anything written with the Null Quill automatically redacts itself from the writer’s memory until it is safely back in a Class M or lower environment.  
Many senior Cartographers carry journals full of pages they can no longer read.

The transactional backbone of multiversal exploration

These three systems govern how entities interact with the Archive, with each other, and with the infinite realities cataloged within.
They are not merely legal frameworks — they are metaphysical instruments that bind stories, timelines, and obligations.

🜂 Contracts
---
(Binding agreements enforced by essence physics)

Contracts in the TRUE Archive are not paper documents.
They are story‑locks: metaphysical agreements that bind participants through causality, Echoes, and Annex‑grade logic.

Types of Contracts

	Causal Contracts  
		Bind actions to consequences across timelines.
		Breaking one causes “causal recoil” — the universe corrects you violently.

	Echo Contracts  
		Agreements written into memory‑space.
		Violation results in memory bleed or identity drift.

	Annex Contracts  
		Forbidden pacts involving Ω‑class entities or spaces.
		Breaking one risks Annex attention.

	Curatorial Contracts  
		Deals made directly with the Curator.
		Terms are always fulfilled — but never as expected.

Contract Structure

	Each contract contains:

	Clause Glyphs 		— symbolic logic defining obligations
	Causal Anchors 		— fixed points ensuring compliance
	Echo Signatures 	— identity markers
	Termination Conditions 	— how the contract ends
	Essence Cost 		— mental strain for maintaining the pact

Contracts are stored in the Lodge of Silent Binding unless they involve external Lodges.

🜃 Bounties
---
(Tasks, hunts, or missions issued by Archive or external powers)

Bounties are tasks: missions that alter or preserve structures.

They are issued by:

	The Curator
	Internal Lodges
	External Lodges
	Entire universes
	Dying timelines

	Annex entities (rare, dangerous)

Bounty Categories
--
	Stabilization Bounties  
		Fix collapsing timelines, seal ruptures, repair logic fractures.

	Extraction Bounties  
		Retrieve artifacts, Echoes, or living records.

	Containment Bounties  
		Neutralize or relocate dangerous entities.

	Annex Bounties  
		Ω‑class missions; success is survival.

	Essence Bounties  
		Intervene to prevent collapse or corruption.

Bounty Components

	Issuer glyph 		— who issued it
	Target Code 		— Annex Code of the reality involved
	Hazard Rating 		— Echo, Divergence, Annex risk
	Reward Structure 	— cycles, permissions, licenses
	Failure Consequences 	— sometimes worse than death

Bounties are posted in the Galleries, Curator’s Office, and external guild halls.

🜄 Offers
--
(Negotiated exchanges of value, knowledge, or access)

Offers are non‑binding proposals that become Contracts only if accepted.
They are the “soft diplomacy” of the Archive.

Offer Types

	Curatorial Offers  
		The Curator provides a clue, artifact, or code in exchange for… something.

	Lodge Offers  
		Internal Lodges trade access, protection, or knowledge.

	Guild Offers  
		External guilds propose joint expeditions, resource trades, or political alliances.

	Reality Offers  
		Entire universes “offer” boons to visitors who fulfill essential roles.

Offer Mechanics

	Non‑binding until sealed with a Contract glyph
	Reversible unless tied to a Causal Anchor
	Negotiable — even with Annex entities
	Time‑sensitive — some expire when timelines shift

Offers are the safest way to interact with unknown powers — but also the easiest to misinterpret.

🜁 Non‑Archive Contracts & Bounties
---
The transactional systems of external Lodges, Guilds, and cosmic powers

Outside the Archive, the multiverse runs on wild law — agreements enforced not by the Curator, but by:

	Causal inertia
	Mythic obligation
	External enforcement guilds
	Reality‑specific metaphysics
	Predatory entities that treat contracts as hunting tools

These systems are older, stranger, and often more dangerous than the Archive’s internal frameworks.

🜂 Non‑Archive Contracts
---
Agreements enforced by external metaphysics, not Archive law

External contracts are not story‑locks like Archive contracts.
They vary wildly depending on the issuing civilization or Lodge.

Major Types of Non‑Archive Contracts

	1. Fate‑Bound Contracts

		Issued by mythic or deterministic universes.
		Breaking one rewrites your destiny.
		Some universes send “Fate Correctors” to enforce terms.

	2. Ledger Contracts (Wandering Ledger)

		Mathematical‑causal agreements.
		Terms are expressed as equations.
		Violations create “causal debt” that must be paid in events.

	3. Forge Contracts (Pale Forge)

		Techno‑mythic fabrication agreements.
		You receive a synthetic universe, tool, or physics constant.
		Payment is usually a memory, lifespan, or timeline.

	4. Witness Contracts (Last Witness)

		End‑of‑world agreements.
		You agree to observe a dying universe.
		Payment is knowledge — or trauma.

	5. Keybreaker Pacts

		Criminal‑grade contracts.
		Terms often involve breaking into sealed realities.
		Enforcement is handled by the Keybreakers themselves.

	6. Shepherd Accords (Horizon Shepherds)

		Humanitarian contracts.
		You escort refugees between worlds.
		Payment is sanctuary, protection, or political favor.

	7. Unwritten Oaths (Unwritten Path)

		Anti‑fate agreements.
		You vow to break a predestined event.
		Failure means becoming part of the fate you tried to break.

🜃 Non‑Archive Contract Structure
---
External contracts use different symbolic systems than Archive glyphs.

Most include:

	Obligation Vector 	— what must be done
	Enforcement Mechanism 	— fate, math, myth, or violence
	Penalty Clause 		— often metaphysical
	Witness Seal 		— who observes the contract
	Termination Condition 	— how the contract ends

Some contracts are alive.
Some contracts hunt you if you break them.

🜄 Non‑Archive Bounties
--
Tasks, hunts, and missions issued by external powers

External bounties are not posted on Archive boards.

They circulate through:

	Guild halls
	Cosmic marketplaces
	Mythic courts
	Rogue universes
	Annex‑adjacent black markets

Major Types of External Bounties

	1. Ledger Reclamation Bounties
		Issued by the Wandering Ledger.

		Retrieve stolen probability.
		Correct impossible events.
		Collect causal debt.

	2. Forge Retrieval Bounties
		Issued by the Pale Forge.

		Recover escaped synthetic universes.
		Neutralize unstable forge‑cores.
		Capture rogue physics.

	3. Witness Terminal Bounties
		Issued by the Last Witness.

		Attend the death of a universe.
		Record its final moment.
		Ensure no one interferes.

	4. Keybreaker Heist Bounties
		Issued by the Keybreakers.

		Break into sealed realities.
		Steal artifacts or entities.
		Evade Archive detection.

	5. Shepherd Evacuation Bounties
		Issued by Horizon Shepherds.

		Evacuate populations from collapsing worlds.
		Stabilize horizon corridors.
		Escort refugees to sanctuary.

	6. Unwritten Intervention Bounties
		Issued by the Unwritten Path.

		Break predestined events.
		Assassinate inevitabilities.
		Destroy causal anchors.

🜅 Non‑Archive Bounty Structure
--
External bounties include:

	Issuer Mark 			— symbol of the Lodge/Guild
	Target Reality Code 	— not Annex Code; external systems vary
	Hazard Tier 			— mythic, causal, temporal, or existential
	Reward Vector 			— cycles, favors, physics, or sanctuary
	Failure Cost 			— often irreversible

Some bounties are predatory:

	Accepting them binds you to the issuer permanently.

🜆 How Non‑Archive Systems Interact With the Archive
---
The Archive does not recognize external contracts.

External Lodges do not respect Archive law.
The Curator may intervene only if Annex risk is involved.
External bounties often conflict with Archive missions.
Some external contracts override Archive contracts (rare, dangerous).
Some external bounties target Archive assets.

This is why the Archive maintains Lodges like:

Curator’s Shadow (counter‑intelligence)
Silent Binding (contract containment)
First Lens (external hazard analysis)
